# Hinjaku (貧弱)

Rust + egui で作られた、軽量で「貧弱」な機能を特徴とする Windows 向け画像ビューアです。

## 主な機能
- **アーカイブ対応**: zip / 7z 内の画像を解凍せずに閲覧可能。
- **マンガモード**: 見開き表示に対応。
- **高速・並列処理**: 4スレッド並列デコードと先読みキャッシュによる快適なページ送り。
- **外部アプリ連携**: As/R (AsrLoad.exe) などの外部ソフトにパスを直接送信。
- **高度なソート**: 名前、日付、サイズ順、および自然順（1, 2, 10...）に対応。
- **ポータブル**: レジストリを汚さず、`config.ini` で設定を完結。
- **単一インスタンス**: 二重起動を防止し、既存のウィンドウでファイルを開き直します。

## 操作方法
| キー / 操作 | 機能 |
|-----------|------|
| 左クリック / `A` | 前のページ |
| `D` | 次のページ |
| `Space` / `M` | マンガモード (1P/2P) 切替 |
| 右クリック | **戻る** |
| ホイール上下 | ページ移動 |
| 右クリック + ホイール | ズーム (拡大・縮小) |
| `S` | 並べ替え設定ウィンドウを表示 |
| `E` | 外部アプリで開く |
| `I` | 画像のスムージング（補正）切替 |
| `Backspace` | エクスプローラーでこのファイルを選択 |
| `Home` / `End` | アーカイブの先頭 / 末尾へジャンプ |
| `PgUp` / `PgDn` | 前 / 次のフォルダ・アーカイブへ移動 |

## ビルド
