# Hinjaku

**吹けば飛ぶよな軽量ビューア**

Hinjaku は、Windows 環境に特化した、極めて軽量で高速な画像ビューアです。  
アーカイブ（ZIP/7z）内の画像を、一時ファイルを作らずに直接ストリーム読み込みすることで、低メモリ消費と高速な閲覧を実現しています。

## 主な特徴

- **Windows 専用設計**: Windows API や `ShellExecuteW` を活用し、OS と親和性の高い動作をします。
- **アーカイブ直接閲覧**: ZIP および 7z 形式に対応。解凍の手間なく中身を閲覧可能です。
- **外部アプリ連携 (ActionKey)**: 表示中のファイルパスを Photoshop などの外部ツールへ即座に渡すことができます。
- **ゼロ・テンポラリ**: アーカイブ展開時にディスクへ一時ファイルを一切書き込みません。
- **INI ベース設定**: 設定はすべて `config.ini` で管理され、ポータブルに使用できます。

## コマンドラインオプション

起動時に以下の引数を指定することで、動作をカスタマイズできます。

```bash
hinjaku.exe [パス] [-c 設定名] [--debug]
```

- **`パス`**: 起動時に開く画像ファイル、フォルダ、またはアーカイブファイルのパスを指定します。
- **`-c`, `--config`**: 使用する設定ファイル（INI）のプロファイル名を指定します。  
  例: `-c sub` と指定すると `sub.ini` を読み込みます。指定がない場合は `config.ini` が使用されます。
- **`--debug`**: コンソールにメモリ使用量やキャッシュ統計などのデバッグログを出力します。

## 主なキー操作

| キー | 操作 |
|:---|:---|
| `<` / `>` (P / N) | 前のページ / 次のページ |
| `Shift` + `<` / `>` | 1枚送り（見開き時） |
| `F` | フィットモード切替（全体 / 幅合わせ / 等倍） |
| `M` / `Space` | マンガモード（見開き表示）の切替 |
| `T` | ディレクトリツリーの表示 / 非表示 |
| `S` | ソート設定ウィンドウの表示 |
| `I` | 画像補間フィルタ（Nearest / Bilinear / Bicubic / Lanczos）の切替 |
| `R` / `Ctrl+R` | 右回転 / 左回転 |
| `BS` (BackSpace) | 現在のファイルをエクスプローラーで表示 |
| `1` ～ `5` | 設定した外部アプリで開く |
| `Esc` | 全画面解除 / ウィンドウを閉じる |
| `Q` | アプリを終了 |

## 使用ライブラリ

本プロジェクトは、以下の素晴らしいオープンソースライブラリを使用して開発されています。

- **eframe / egui**: GUI フレームワーク
- **image**: 画像デコード処理
- **zip-rs**: ZIP アーカイブ操作
- **sevenz-rust**: 7z アーカイブ操作
- **rust-ini**: INI 設定ファイルの読み書き
- **windows-sys**: Windows API 連携
- **rfd**: ネイティブファイルダイアログ

## 開発憲法 (Immutable Rules)

1. **Windows Native Only**: Windows 環境に特化し、UNC パスや OS 固有機能を優先する。
2. **INI-Based Config**: 設定は INI 形式を厳守する。JSON/TOML は採用しない。
3. **Zero-Temp Strategy**: ディスクへの一時ファイル作成を禁止し、メモリ内で完結させる。
4. **No-Code Friendly**: 初心者でも理解しやすい具体的な命名と日本語コメントを維持する。
