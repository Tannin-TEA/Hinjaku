# Hinjaku - Image Archive Viewer

Rust/egui で構築された、軽量で高速な画像アーカイブビューアです。

## 主な機能
- **対応フォーマット**: ZIP, 7z, 通常のフォルダ, 単一画像ファイル。
- **対応画像形式**: JPG, PNG, WEBP, GIF, BMP, TIFF。
- **表示モード**:
  - 1ページ表示 / マンガモード（2ページ見開き）。
  - 右開き (RTL) / 左開き対応。
  - フィット表示 / 等倍・ズーム表示。
  - フルスクリーン / ボーダレス最大化。
- **ナビゲーション**:
  - サイドバーによるディレクトリツリー表示。
  - フォルダ間移動（前/次のフォルダへ）。
  - 自然順（数値考慮）ソートを含む多角的なソート機能。
- **外部連携**: 任意の外部アプリで表示中の画像を開く機能。

## 技術仕様・内部パラメータ
スムーズなユーザー体験（UX）を維持するため、以下の仕様が実装されています。

### 1. ナビゲーション・ガード (誤操作防止)
ページめくり時の「意図しない連打」を防ぐため、以下のロック時間が設定されています。
- **フォルダ移動ロック**: `2.0秒` (新しいフォルダ/アーカイブを開いた際、最初の画像が表示されてからロック)
- **ページ同期ロック**: `0.05秒` (ページ移動やマンガモードの左右同期が完了した瞬間のロック)
- **マウスホイールしきい値**: `40.0` (感度調整用。一定量のスクロールが蓄積されるまでページ移動しません)

### 2. パフォーマンスとメモリ管理
- **画像キャッシュ**: 最大 `13枚`。
- **プリフェッチ**: 現在のページから前方 `5枚`、後方 `5枚` をバックグラウンドで事前読み込み。
- **自動リサイズ**: 解像度が `4096px` を超える画像は、GPU転送前に自動的にダウンスケールされます。
- **アニメーション制限**: GIF/WebP アニメーションは、ファイルサイズが `30MB` 以下のもののみ再生されます。
- **スタッター防止**: GPUへのテクスチャ転送は1フレームにつき最大 `2枚` に制限し、操作の引っかかりを防止します。

### 3. マンガモードのインテリジェンス
- 見開き画像（横長画像）を検出すると、自動的に1枚で表示します。
- フォルダの1枚目（表紙想定）は常に1枚で表示し、2枚目から見開きを開始します。
- 左右のページが両方ロードされるまで表示更新を待機し、ガタつきを防止します。

## キーボードショートカット

| キー | アクション |
|:---|:---|
| `Arrow Right` / `N` | 次のページ / 次の見開き |
| `Arrow Left` / `P` | 前のページ / 前の見開き |
| `Arrow Down` / `Arrow Up` | (マンガモード時) 1ページずつ進む/戻る |
| `M` / `Space` | マンガモードの切り替え |
| `Y` | 右開き/左開きの切り替え |
| `Enter` | フルスクリーン切り替え |
| `Alt + Enter` | ボーダレス最大化切り替え |
| `F` | フィット表示の切り替え |
| `+` / `-` | 拡大 / 縮小 |
| `R` | 右回転 (90度) |
| `Ctrl + R` | 左回転 (-90度) |
| `S` | 並べ替え設定ウィンドウを表示 |
| `T` | ディレクトリツリーを表示 |
| `I` | 画像補正（スムージング）の切り替え |
| `PgDn` / `PgUp` | 次のフォルダ / 前のフォルダへ移動 |
| `Backspace` | ファイルをエクスプローラーで表示 |
| `E` | 設定された外部アプリで開く |
| `Q` / `Ctrl + W` | アプリを終了 |
| `Esc` | 全画面解除 / ウィンドウを閉じる |

## コマンドライン引数
```bash
hinjaku.exe [画像またはフォルダのパス] [-c config.ini]
```

## 設定 (config.ini)
以下の項目が `config.ini` (TOML形式) で保存されます。
- `external_app`: 外部アプリのパス。
- `external_args`: 引数（`%P` がファイルパスに置換されます）。
- `linear_filter`: 画像補正のデフォルト設定。
- `allow_multiple_instances`: 複数起動の許可設定。
- `sort_mode` / `sort_order` / `sort_natural`: ソート設定の永続化。
- `manga_rtl`: 右開きのデフォルト設定。
