# Hinjaku (貧弱)

Rust + egui で作られた、軽量かつ高速な Windows 向けアーカイブ・画像ビューアです。

## 主な機能
- **高速・並列処理**: 4スレッド並列デコードと先読みキャッシュ、GPUサンプリング活用による爆速なページ送り。
- **アーカイブ対応**: ZIP / 7z 内の画像を解凍せずに直接閲覧可能。
- **マンガモード**: 見開き表示（2P）に対応。右開き（RTL）と左開き（LTR）を切り替え可能。
- **シームレス移動**: フォルダの終端に達すると、階層を遡って隣接するディレクトリやアーカイブへ自動で移動します。
- **外部アプリ連携**: As/R (AsrLoad.exe) などの外部ソフトにパスを送信。アーカイブ内ファイルも仮想パスで連携可能。
- **高度なソート**: ファイル名、更新日時、サイズによる昇降順、および自然順（1, 2, 10...）に対応。
- **ポータブル**: レジストリを一切使用せず、設定は実行ファイルと同階層の `config.ini` に保存されます。
- **単一インスタンス**: 二重起動を防止。後から開いたファイルを既存のウィンドウへ転送し、自動でフォーカスします。

## 操作方法
| カテゴリ | 操作 | 機能 |
|---|---|---|
| **ページ移動** | `←` / `P` / 左クリック | 前のページ / 前の見開きへ |
| | `→` / `N` | 次のページ / 次の見開きへ |
| | `↑` / `↓` | 1ページ単位で戻る / 進む (見開き位置の微調整用) |
| | `Home` / `End` | アーカイブ・フォルダの最初 / 最後へジャンプ |
| | ホイール上下 | ページ送り (1ノッチ1移動に最適化) |
| | 右クリック | **戻る** (ページを戻す) |
| **フォルダ移動** | `PgUp` / `PgDn` | 前 / 次のフォルダ・アーカイブへ移動 |
| **ズーム** | 右クリ長押し + ホイール | 拡大 / 縮小 |
| | `+` / `-` | 拡大 / 縮小 |
| | `F` | フィット表示 (全体表示) 切り替え |
| **表示設定** | `Space` / `M` | マンガモード (1P/2P) 切り替え |
| | `Y` | 右開き (RTL) / 左開き (LTR) 切り替え |
| | `I` | 画像のスムージング (線形補完) 有効 / 無効切り替え |
| | `R` / `Ctrl`+`R` | 右回転 / 左回転 |
| **全画面** | `Enter` | 全画面切替 (枠あり最大化) |
| | `Alt` + `Enter` | ボーダレス全画面切替 (枠なし最大化) |
| | `Esc` | 全画面・ボーダレス・設定画面を解除 |
| **連携・設定** | `S` | 並べ替え設定を表示 (矢印キーで選択、Enterで確定) |
| | `E` | 外部アプリで開く (設定されたソフトへパスを送信) |
| | `Backspace` | エクスプローラーで現在のファイルを選択して開く |

## 外部アプリ連携の設定
アプリ内の「ファイル」→「外部アプリ設定」から設定できます。
- **As/R 連携例**:
  - アプリパス: `C:\Tools\Asr\AsrLoad.exe`
  - コマンド引数: `/jf %P`
  - ※ `%P` は表示中のファイルパス（アーカイブ内なら `zip名\内部パス`）に置換されます。
  - ※ パスにスペースが含まれる場合は、自動的にダブルクォーテーションで囲まれます。

## 技術仕様
- **通信**: 単一インスタンス制御のため、ローカルの TCP ポート `43210` を使用します。
- **アイコン**: 独自の「H」アイコンをプログラム実行時に生成して適用します。
